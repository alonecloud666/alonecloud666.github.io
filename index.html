接下来，我们把代码上传到Github上面

（注意：在Git中Ctrl+Insert是复制，shift+Insert是粘贴，也可以右键）

第一步：进入自己项目文件夹，右键点击Git Bash

第二步：输入git init

第三步：输入git add .

第四步：输入git config –global user.name”你的名字”和git config –global user.email”你的邮箱”


第五步：输入git commit -m “first commit”，表示对这次提交的注释

第六步：输入git remote add origin https://自己的仓库url地址，这步可能会出现错误，如图

这时候就要输入$ git remote rm origin，然后再执行添加即可

第八步：输入git push -u origin master，可能要输入用户名和密码，输入即可
至此，代码已经上传到Github上面啦
接下来，安装nodejs

配置SSH Key
输入以下代码：
ssh-keygen -t rsa -C “your_email@example.com“

系统会在/Users/your_user_directory/.ssh/x下生成私有密钥id_rsa和公开密钥id_rsa.pub

用记事本打开id_rsa.pub，里面有一大串字符，将它复制
然后我们在GitHub上面添加本地SSH密钥信息，在右上角选择Settings进入设置界面
点击SSH and GPG keys，再点击New SSH key
Title可以随便填，这里我命名为github pages
Key里面填刚刚复制的Key
最后点击Add SSH key
就搞定啦

接下来，我们来安装hexo插件
